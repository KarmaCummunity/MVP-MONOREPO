## 📋 7. כללים עסקיים (MVP)

> מסמך זה מאחד את הכללים העסקיים החיוניים ל-MVP בלבד. כל כלל הוא **מקודד**: `R-MVP-{תחום}-{מספר}`. החזון הרחב כולל כללים מורכבים יותר ([`../PRD_HE_V2/07_Business_Rules.md`](../PRD_HE_V2/07_Business_Rules.md)) שאינם רלוונטיים ל-MVP.

---

### 7.1 כללי ליבה כלליים

| מזהה             | כלל                                                                                              |
| ---------------- | ------------------------------------------------------------------------------------------------ |
| **R-MVP-Core-1** | **חינם תמיד.** אסור שיועבר כסף, חליפין, או "השתתפות בהוצאות" בין משתמשים. הפרה = הסרת פוסט וחסימת חשבון. |
| **R-MVP-Core-2** | **משתמש חייב להיות רשום** כדי לבצע כל פעולה. אורח רואה רק 3 פוסטים, ואז Overlay הצטרפות.            |
| **R-MVP-Core-3** | **גיל מינימום: 18.** משתמש מצהיר על גילו בהרשמה. (ב-MVP אין אימות גיל; הצהרה משפטית בלבד).        |
| **R-MVP-Core-4** | **שפה: עברית בלבד.** האפליקציה רצה ב-RTL. אנגלית תיתמך בגרסה עתידית. ארכיטקטורת i18n מההתחלה.       |
| **R-MVP-Core-5** | **גיאוגרפיה: ישראל בלבד.** רשימת הערים מוגבלת לערים בישראל.                                       |
| **R-MVP-Core-6** | משתמש יכול להחזיק **חשבון אחד בלבד** (לפי טלפון / מייל / Google ID). אין חשבונות כפולים.            |
| **R-MVP-Core-7** | **רב-פלטפורמיות חובה:** iOS + Android + Web. עיצוב רספונסיבי לכל גודל מסך (מובייל קטן עד דסקטופ גדול). ראה [`06_Navigation_Structure.md#6.6`](./06_Navigation_Structure.md). |
| **R-MVP-Core-8** | **3 שיטות אימות:** Google SSO / טלפון OTP / מייל + סיסמה. אין שילוב חשבונות ב-MVP. ב-iOS חובה גם Apple SSO (App Store Guidelines). |
| **R-MVP-Core-9** | **אין וי כחול / אימות זהות מורחב.** כל משתמש שאימת אחד משלוש שיטות = משתמש מלא.                       |

### 7.2 כללי פוסטים ופרסום

| מזהה               | כלל                                                                                                       |
| ------------------ | --------------------------------------------------------------------------------------------------------- |
| **R-MVP-Items-1**  | **תמונה: חובה ב"לתת"; אופציונלית ב"לבקש".** במצב "לבקש" ללא תמונה – הפיד מציג אייקון קטגוריה ברירת מחדל.   |
| **R-MVP-Items-2**  | **קטגוריה חובה.** הפוסט חייב להיות משויך לאחת מ-10 הקטגוריות:                                              |
|                    | רהיטים / בגדים / ספרים / משחקים / ציוד תינוקות / מטבח / ספורט / חשמל / כלי עבודה / אחר                     |
| **R-MVP-Items-3**  | **בעל הפוסט הוא היחיד** שיכול לערוך, למחוק, או לסמן "נמסר".                                                |
| **R-MVP-Items-4**  | **סגירת פוסט (נמסר):** הפוסט נסגר רק אחרי שהחפץ **נמסר בפועל** (לא אחרי תיאום או "שידוך"). שני מסלולי סגירה: |
|                    | * **עם סימון מקבל** → סטטוס `closed_delivered`, נשמר תמיד אצל הבעל ואצל המקבל.                              |
|                    | * **ללא סימון מקבל** → סטטוס `deleted_no_recipient`, נמחק לחלוטין תוך 7 ימים.                              |
| **R-MVP-Items-5**  | **משך פוסט פעיל ברירת מחדל: 60 ימים.** לאחר מכן הפוסט עובר אוטומטית לסטטוס `expired`. המשתמש מקבל התראה 7 ימים לפני. |
| **R-MVP-Items-6**  | **פתיחה מחדש של פוסט סגור (Reopen):** הבעל יכול לפתוח מחדש פוסט בכל עת. אם נסגר עם סימון מקבל – המקבל מקבל התראה שהסימון בוטל. הסטטיסטיקות מתעדכנות בהתאם. |
| **R-MVP-Items-7**  | **5+ פתיחות מחדש לפוסט אחד = Suspect**. הפוסט עולה לבדיקת אדמין.                                            |
| **R-MVP-Items-8**  | **מקסימום פוסטים פעילים בו-זמנית: 20 לכל משתמש.** למניעת ספאם.                                              |
| **R-MVP-Items-9**  | **אסור פרסום פריטים מסוכנים:** כלי נשק, חומרים מסוכנים, סמים, פריטים לא חוקיים, חיות חיות.                  |
| **R-MVP-Items-10** | **אסור פרסום פריטים שהם בפועל למכירה.** אם זוהה בפוסט מחיר/השתתפות → דיווח אוטומטי + הסרה.                   |
| **R-MVP-Items-11** | **כל פוסט מציג חובה:** תג ברור 🎁 לתת / 🔍 לבקש. אסור פוסט בלי טוגל מפורש.                                    |
| **R-MVP-Items-12** | **רמת חשיפת פוסט:** המפרסם בוחר בעת יצירה: 🌍 פומבי (ברירת מחדל) / 👥 רק עוקבים שלי. **לאחר פרסום ניתן רק להעלות חשיפה** (עוקבים → פומבי), לא להוריד. |
| **R-MVP-Items-13** | **כתובת מלאה חובה למילוי בכל פוסט** (עיר + רחוב + מספר). **רמת חשיפת המיקום בפיד נבחרת ע"י המפרסם** – ראה R-MVP-Privacy-1. |

### 7.3 כללי בטיחות ופרטיות

| מזהה                 | כלל                                                                                                                            |
| -------------------- | ------------------------------------------------------------------------------------------------------------------------------ |
| **R-MVP-Privacy-1**  | **כתובת מלאה חובה למילוי**, אבל **רמת ההצגה נבחרת ע"י המפרסם:** עיר בלבד / עיר + רחוב (ברירת מחדל) / כתובת מלאה. הכלל זהה ל"לתת" ו-"לבקש". |
| **R-MVP-Privacy-2**  | **אין גילוי טלפון / מייל בפיד או בפרופיל הציבורי.** התקשרות מתבצעת רק בצ'אט הפנימי.                                              |
| **R-MVP-Privacy-3**  | **חסימת משתמש** = המשתמש החסום לא רואה את הפוסטים שלי, ולא יכול לשלוח לי הודעות. הוא **לא יודע** שהוא חסום.                       |
| **R-MVP-Privacy-4**  | **דיווח על תוכן** – זמין מכל פוסט / פרופיל / שיחה. כל דיווח נשלח כהודעת מערכת לחשבון הסופר אדמין (`karmacommunity2.0@gmail.com`). |
| **R-MVP-Privacy-5**  | **3 דיווחים על אותו פריט (פוסט / משתמש / תגובה) = הסרה אוטומטית.** הסופר אדמין מקבל הודעה ויכול לשחזר אם מצא תקין.                |
| **R-MVP-Privacy-6**  | **מחיקת חשבון** = מחיקת כל הפוסטים שלי, השיחות (מצדי), והפרופיל. אסור לשחזר.                                                    |
| **R-MVP-Privacy-7**  | **אסור איסוף מידע מעבר לנדרש לפעולת המוצר.** הסכמת המשתמש לתנאי שימוש מקצרת ושקופה.                                              |
| **R-MVP-Privacy-8**  | **שמירת סינון אחרון:** סינון/חיפוש שהמשתמש הפעיל נשמר מקומית (לוקאלית) ובסשן הבא נטען אוטומטית. ניתן לאיפוס דרך "נקה הכל".         |
| **R-MVP-Privacy-9**  | **שינוי רמת חשיפת פוסט** לאחר פרסום: רק להעלאה (עוקבים → פומבי). לא ניתן להוריד (פומבי → עוקבים).                                 |
| **R-MVP-Privacy-10** | **5 דיווחים שווא ברצף ע"י משתמש** → השעיה זמנית של חשבון המדווח (למניעת ניצול לרעה).                                            |

| מזהה             | כלל                                                                                                                |
| ---------------- | ------------------------------------------------------------------------------------------------------------------ |
| **R-MVP-Chat-1** | **שיחות 1-על-1 בלבד.** אין צ'אט קבוצתי ב-MVP.                                                                       |

### 7.7 הכרזות אסורות

* **תוכן המסית, מדיר, גזעני, או פוגעני** – הסרה מיידית + חסימת מפרסם.
* **הצעת מוצר במחיר / חליפין** – הסרה מיידית.
* **שיתוף פרטי משתמש שאינם פומביים** ללא הסכמה – הסרה + חסימה.
* **פרסומות מסחריות / קישורים שיווקיים** בפוסטים – הסרה.
* **תוכן פורנוגרפי / מיני מפורש** – הסרה + חסימה + דיווח לרשויות.

### 7.9 תקנון משתמש מקוצר ל-MVP

המשתמש מאשר בעת ההרשמה:

> 1. אני מתחייב לפרסם תוכן אמיתי ומדויק.
> 2. אסור לי להציע או לבקש תשלום בפלטפורמה.
> 3. אני מאשר שגילי 18+.
> 4. הפלטפורמה אינה אחראית על העסקאות שאעשה דרכה. אני אחראי על הבטיחות של עצמי.
> 5. אני מסכים לכך שהמייל/טלפון שלי יישמר במאגר הפלטפורמה (לא ייחשף בפומבי).
> 6. תוכן בעייתי שיתפרסם על ידי – יוסר מיידית, וייתכן וחשבוני יושעה.
